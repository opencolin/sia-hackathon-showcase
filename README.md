# SIA Agents Hackathon — Project Showcase

A static showcase of the **19 projects** built at the *Self-Improving AI (SIA) Agents — Emergency Hackathon* (AGI House SF × Nebius, June 6 2026).

Each project builds on the **SIA framework** (Self-Improving AI with Harness & Weight Updates, Hebbar et al. 2026) — a language-model agent that rewrites its own harness and/or weights in a propose → run → score → keep loop.

- **Event:** https://www.getoatmeal.com/e/self-improving-ai-sia-agents-emergency-hackathon-by-agi-house-sf-nebius
- **SIA framework:** https://github.com/hexo-ai/sia

## Run locally
It's a single self-contained `index.html` — just open it, or:
```bash
python3 -m http.server 8000   # then visit http://localhost:8000
```

## Deploy
Zero-config static deploy (Vercel, GitHub Pages, Netlify). On Vercel: `vercel --prod`.

Project descriptions are the builders' own; attribution is by public GitHub handle. Two repos (SIA-improvement, Startup Wedge) were private at showcase time.
